v-echarts-pie.js 使用方式

 

描述:
    圓餅圖
維度量值:
    維度1個，量值1個；dt維度是分顏色，量值決定大小
html:
    id:必填，元件id，要有唯一性
屬性:
    data : [{name:'abc', value:44},....]
    colors : 色碼["#xxxxx","#xxxxx"]
    showlabel : 字串"true", 預設"false", 設true的話會顯示label
    hiddenarr : array，將要隱藏的name放在array內
參考:
    https://echarts.apache.org/examples/zh/editor.html?c=pie-legend

Source Code

                
                <v-echarts-pie 
                    id="gender_chart" 
                    showtext='元'
                    :data="datalist"
                    :colors= "['#8469b3','#31bdba','#c0d95f','#42bd56']"
                    :hiddenarr="['小新']"
                    :plotheight="500"
                    valuelabelunit="%">
                </v-echarts-pie>
