v-linear-regression.js 使用方式

 

描述:
    線性迴歸分析圖
維度量值:
    維度1個，量值1個
限制:
    data裏面一定要有dt, dt格式一定'yyyy-mm-dd'
html:
    id:必填，元件id，要有唯一性
屬性:
    data : [{dt:'2020-10-01', 總銷售額:44},....]
參考:
    https://echarts.apache.org/examples/zh/editor.html?c=scatter-linear-regression
依賴:
    https://github.com/ecomfe/echarts-stat/blob/master/dist/ecStat.js

Source Code

                
                <v-linear-regression 
                    id="plot2"
                    :data='datalist'
                    :y_label_format="(val, key) => {
                        if (key=='總銷售額') return '$' + val
                        else return val
                    }">
                </v-linear-regression>
