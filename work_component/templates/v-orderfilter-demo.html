v-linear-regression.js 使用方式

 

描述:
    包含「付費方式」、「購買方式」、「門店」、「日期區間」的複合元件
    會自動儲存「付費方式」、「購買方式」、「門店」的值
    「日期區間」自動選擇過去七天
維度量值:
    維度1個，量值1個
html:
    id:必填，元件id，要有唯一性
屬性:
    opts : 預設下拉單的內容
    {
        ord_paytype:[]
        buytype:[]
        store:[]
    }

事件：
　　onchange(val)當回傳值發生改變時觸發
        - val :目前選單的選擇
        {
            ord_paytype:[],付費類型
            buytype:[],購買方式
            store:[],門店
            dr:"" 區間
        }
    oncommit()使用者按下搜尋計算按鈕時觸發
            
參考:
    https://codertw.com/%E5%89%8D%E7%AB%AF%E9%96%8B%E7%99%BC/197439/
依賴:
    vue element
    vue materaial
    v-el-select-multi元件
